# \# GRC Risk Assessment Lab

# 

# This project shows a complete workflow from technical vulnerability scanning to risk analysis in a GRC context.

# 

# The lab was carried out using Metasploitable 2 as the target system, with the goal of identifying vulnerabilities and explaining what they mean from a business and risk perspective.

# 

# \## Structure

# 

# \- \*\*documentation/\*\* → full report (analysis, risk, business impact)

# \- \*\*technical-evidence/\*\* → raw scan outputs (nmap, nikto, nuclei)

# \- \*\*screenshots-media/\*\* → screenshots from the lab

# 

# \## Key Focus

# 

# \- Technical scanning (Nmap, Nikto, Nuclei)

# \- Risk assessment and analysis

# \- Financial impact (SLE \& ALE)

# \- Business Impact Analysis (BIA)

# \- Control mapping (NIST, CIS)
